# Projeto Banco de Dados - Farmácia 💊

## Descrição do Sistema
Este sistema visa gerenciar os processos operacionais de uma farmácia, incluindo o controle de estoque de medicamentos, cadastro de clientes, registro de receitas médicas para remédios controlados e gestão de vendas e fornecedores..
